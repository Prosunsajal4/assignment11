# 📚 BookCourier

### Library-to-Home Book Delivery System (Assignment 11)

**BookCourier** is a professional front-end web application that simulates a library-to-home book delivery service.  
The project emphasizes a clean, modern UI, fully responsive layout, and user-centric design, suitable for academic submission and portfolio showcase.

🌐 **Live Website:** [https://assignement11-8c757.web.app/](https://assignement11-8c757.web.app/)

---

## 📝 Project Overview

**Last updated:** 2026-04-29

BookCourier allows users to explore a book delivery service through a library-style interface.  
The project demonstrates:

- Modern front-end design practices
- Responsive layouts for desktop, tablet, and mobile
- Simple client-side interactivity using Vanilla JavaScript
- Clean, organized project structure

This project was developed as **Assignment 11** to showcase practical front-end development skills.

---

## 🎯 Project Objectives

- Create an intuitive and user-friendly book delivery interface
- Apply responsive web design principles
- Practice modern front-end techniques using Tailwind CSS
- Deploy a live, accessible version for demonstration
- Produce an academic-ready project with professional documentation

## 🌟 Key Features

- 📦 Library-to-home book delivery concept
- 📱 Fully responsive design across all devices
